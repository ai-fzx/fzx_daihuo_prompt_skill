# fzx-daihuo-prompt-skill

> **风之馨技术录** 出品 — 电商带货九宫格 AI 提示词生成技能

## 功能

上传商品图片（支持多图），自动生成：

- 🖼️ **GPT-Image 2 九宫格生图提示词** — 9 张，覆盖白底主图/模特上身/细节特写/场景图/平铺搭配/品牌氛围
- 🎬 **Seedance 2.0 带货视频提示词** — 3 条，覆盖商品旋转展示/模特动态上身/生活场景

## 快速开始

### 方式一：在 WorkBuddy 中使用（推荐）

1. 打开 WorkBuddy，进入项目
2. 上传商品图片
3. 输入：`帮我生成带货提示词` / `九宫格提示词` / `GPT-Image2 提示词`
4. AI 自动触发此技能，输出完整提示词

### 方式二：脚本方式

```bash
node scripts/generate.js ./product.jpg
node scripts/generate.js --output ./output.md ./product.jpg
# 多图模式
node scripts/generate.js ./product-front.jpg ./product-detail.jpg
```

## 九宫格图类型

| # | 类型 | 用途 |
|---|------|------|
| 1 | 白底商品主图 | 淘宝/京东主图 |
| 2 | 模特正面上身（街头） | 小红书/抖音 |
| 3 | 模特背面（版型展示） | 小红书/抖音 |
| 4 | 核心细节特写 A | 电商详情页 |
| 5 | 核心细节特写 B | 电商详情页 |
| 6 | 自然场景上身图 | 小红书/种草 |
| 7 | 城市/运动场景图 | 抖音/带货 |
| 8 | 平铺搭配氛围图 | 小红书九宫格 |
| 9 | 品牌高级感封面 | 抖音封面 |
