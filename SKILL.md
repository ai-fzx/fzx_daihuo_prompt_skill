# fzx-daihuo-prompt-skill — 带货九宫格提示词生成技能

## Overview

分析用户提供的商品图片，自动识别商品类目、视觉特征、材质、风格，生成：
1. **GPT-Image 2 九宫格生图提示词**（9 张，覆盖白底/上身/细节/场景/氛围等维度）
2. **Seedance 2.0 视频提示词**（3 条，覆盖商品展示/模特上身/生活场景三类视频）

## When to Use

- 用户上传商品图片，要求生成带货内容提示词
- 用户提到"九宫格"、"GPT-Image2"、"Seedance"、"带货视频"关键词
- 用户需要将商品图转化为可用于 AI 生图/生视频的结构化提示词
- 用户想做小红书/抖音/淘宝的 AI 带货内容

## 工作流程

### Phase 1：图片分析（必须完成，否则提示词无效）

仔细分析图片，提取以下维度：

| 维度 | 分析要点 |
|------|---------|
| **商品类目** | 上衣/裤子/鞋/包/配饰/家居/数码等 |
| **主色调** | 主色+辅色+点缀色（精确到色号描述） |
| **印花/图案** | 纯色/条纹/格子/印花类型（水墨/碎花/几何等） |
| **版型/形态** | 宽松/修身/A型/H型等版型描述 |
| **面料质感** | 光泽感/哑光/绒面/冰丝/牛仔/针织等 |
| **核心细节** | 领口/袖型/拉链/扣子/口袋/logo/绣花等 |
| **风格定位** | 街头/运动/商务/度假/复古/极简等 |
| **目标人群** | 性别+年龄段（如：20-30岁女性/青年男性） |

### Phase 2：生成 GPT-Image 2 九宫格提示词

必须覆盖以下 9 种图类型（顺序固定）：

```
图1：纯白底商品主图（平铺或隐形模特）
图2：男/女模正面上身展示（街头/日常场景）
图3：男/女模背面展示（展示版型/剪裁）
图4：核心细节特写A（最具卖点的工艺细节）
图5：核心细节特写B（第二卖点细节）
图6：自然场景上身图（户外/海边/公园等）
图7：城市/运动场景上身图（行走/运动状态）
图8：平铺搭配图（商品+周边单品+场景道具）
图9：品牌氛围感图（高级感/品牌调性封面）
```

**每条 GPT-Image 2 提示词格式**：
```
A [商品英文描述，含关键视觉特征]. [模特/场景描述]. [动作/状态]. 
[画面构图]. [光线描述]. [风格关键词: e-commerce/editorial/lifestyle/product]. 
[画质描述: ultra-detailed/sharp focus/high resolution].
```

**关键规则**：
- 全部用英文撰写（GPT-Image 2 英文提示词效果更好）
- 模特必须指定人种+年龄段（如 young Asian female, age 20-25）
- 商品描述必须包含：颜色+材质+款式+最少2个核心细节
- 每条 80-150 词，不宜过短或过长

### Phase 3：生成 Seedance 2.0 视频提示词

必须生成 3 条，分别对应：

```
视频1：商品静态展示视频（旋转/飘动/产品特写运镜）
视频2：模特上身动态展示（行走/动作展示，强调衣物动态）
视频3：生活场景带货视频（有故事感，氛围感强，高转化）
```

**每条 Seedance 提示词格式**（四段式）：
```
[Scene] 场景人物物品描述（英文）
[Motion] 镜头运动+人物动作描述（英文）
[Style] 画面风格+色彩+帧率描述（英文）
[Mood] 情绪/氛围关键词（英文）
```

**Seedance 视频提示词关键规则**：
- 每条 60-120 词
- [Motion] 段必须描述具体的镜头语言（dolly in/pull back/tracking shot/slow motion 等）
- 面料飘动/动态效果要单独描述（体现商品质感）
- [Mood] 段对齐商品目标人群的情感诉求

### Phase 4：输出格式

```markdown
## 商品分析结果
[表格展示 Phase 1 的 8 个维度分析]

## GPT-Image 2 九宫格提示词

### 图1 — [类型名称]
[提示词]

... （共9条）

## Seedance 2.0 视频提示词

### 视频1 — [类型名称]
[提示词]

... （共3条）

## 使用建议
[平台 × 用途 × 推荐图/视频 的对应表格]
```

## 多图处理规则

- 用户提供多张图时：先分析所有图，识别是否为同一商品（不同颜色/角度），还是不同商品
- **同一商品多角度**：合并分析，提示词更丰富准确
- **不同商品**：询问用户"是否分开生成？"，默认每件商品独立输出一套提示词

## 质量检查清单

生成完成后自查：
- [ ] 9 张图类型是否覆盖全（白底/正背面/细节×2/场景×2/平铺/氛围）
- [ ] 每条提示词是否包含商品颜色+材质+核心细节
- [ ] 模特描述是否指定了人种和年龄段
- [ ] Seedance 提示词是否有 [Motion] 具体镜头语言
- [ ] 是否有平台使用建议表格

## Common Mistakes

| 错误 | 正确做法 |
|------|---------|
| 提示词描述过于笼统（"a pair of shorts"） | 必须包含完整特征（颜色+材质+印花+细节） |
| 忘记指定模特特征 | 始终指定 young Asian male/female, age XX-XX |
| Seedance 提示词没有镜头语言 | [Motion] 段必须有 dolly/tracking/slow motion 等 |
| 只生成了7或8张图 | 严格生成9张，类型不重复 |
| 提示词用中文 | GPT-Image 2 和 Seedance 均用英文 |
