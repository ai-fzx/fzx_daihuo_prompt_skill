# fzx-daihuo-prompt-skill

> **风之馨技术录** 出品 — 电商带货九宫格 AI 提示词生成技能

## 功能

上传商品图片（支持多图），自动生成：

- 🖼️ **GPT-Image 2 九宫格图片提示词** — **1条**，一次性生成一张 3×3 布局的九宫格大图
- 🎬 **Seedance 2.0 带货视频提示词** — 3条，覆盖商品展示/模特动态/生活场景

## 核心概念

| 旧方案（错误） | 新方案（正确） |
|---------------|--------------|
| 生成 9 条独立提示词 | **1 条完整提示词**生成一张 3×3 九宫格大图 |
| 9 张分散图 | **1 张大图**，9 格合一 |

## 九宫格布局

```
┌──────────────┬──────────────┬──────────────┐
│  ① 白底主图   │  ② 模特正面   │  ③ 模特背面   │
├──────────────┼──────────────┼──────────────┤
│  ④ 细节特写A │  ⑤ 平铺搭配   │  ⑥ 细节特写B │
├──────────────┼──────────────┼──────────────┤
│  ⑦ 场景上身  │  ⑧ 场景上身   │  ⑨ 品牌氛围   │
└──────────────┴──────────────┴──────────────┘
```

## 快速开始

### 方式一：在 WorkBuddy 中使用（推荐）

1. 打开 WorkBuddy，上传商品图片
2. 输入：`九宫格提示词` / `带货九宫格` / `GPT-Image2九宫格`
3. AI 自动触发技能，输出 **1条九宫格提示词** + **3条视频提示词**

### 方式二：脚本方式

```bash
node scripts/generate.js ./product.jpg
node scripts/generate.js --output ./output.md ./product.jpg
# 多图模式
node scripts/generate.js ./product-front.jpg ./product-detail.jpg
```

## 适用场景

| 平台 | 内容类型 | 提示词来源 |
|------|---------|----------|
| 小红书 | 九宫格封面（9张图） | 九宫格提示词 → 截图裁成9张 |
| 抖音 | 单张大封面 | 九宫格提示词 → 1:1裁剪 |
| 抖音/快手 | 带货短视频 | Seedance 视频提示词 × 3 |
| 天猫/京东 | 主图视频 | 视频1（商品展示） |

## 提示词格式

**GPT-Image 2 九宫格**：单条完整英文提示词，包含全局设定 + 9个格子的具体描述，300-500词

**Seedance 视频**：四段式（Scene / Motion / Style / Mood），每条 60-120 词
